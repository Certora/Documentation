Pragmas
=======

Syntax
------
```{versionchanged} 2.0
Pragmas have been {ref}`removed <cvl2-removed-pragma>`.
```
The syntax for the `pragma` statement is given by the following [EBNF grammar](syntax):

```
pragma ::= "pragma" "specify" number "." number
```
